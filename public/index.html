Scheduling + context

予定を決めるだけでなく、会う前から話しやすく。

キマルは、日程調整と事前アンケートをひとつにした予約ツールです。空いている時間を選んでもらうだけでなく、話したい内容や今の目標も事前に共有できます。

初対面の壁を低くして、限られた面談時間を本題から始めやすくします。

無料で始める予約ページを見る
Free: 3ヶ月先まで / Pro: 6ヶ月先まで

無料でも、予約受付に必要な基本機能を。

30分・45分・60分の予約時間、前後バッファ、曜日ごとの受付時間、事前アンケートに対応。予約受付の条件を管理画面からまとめて設定できます。

管理画面を開くアカウント作成

Features

会う前の準備まで整える日程調整。

予約時間設定

30分・45分・60分から選択。面談、相談、オンライン打ち合わせなど、用途に合わせて設定できます。

前後バッファ

前後に15分・30分の余白を設定できます。移動時間や準備時間を含めた予約受付が可能です。

公開期間制御

無料版は3ヶ月先まで、有料版は6ヶ月先まで予約枠を公開できます。

開催方法

対面、Google Meet、電話、カスタムURL、後で連絡に対応。Zoomは将来対応を想定した設計です。

事前アンケート

無料版は2問、有料版は5問まで設定可能。会う前に、話したいことや共有事項を確認できます。

招待コード

対象ユーザーは招待コードを入力することで、有料版機能を利用できます。コードは公開予約ページには表示されません。
