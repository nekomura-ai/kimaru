キマル（以下「本サービス」）における個人情報の取り扱いについて定めます。本文は法務確認のうえ正式版を掲載します（ドラフト）。

1. 取得する情報

氏名、メールアドレス、予約情報、事前アンケートの回答、（任意で）生年月日、Googleカレンダー連携により取得する空き時間・予定情報等。

2. 利用目的

本サービスの提供（日程調整、通知メールの送信、面談前後の管理）、本人確認、サポート対応、サービス改善のために利用します。

3. Googleユーザーデータの取り扱い

Googleカレンダーのデータは、空き枠の判定と予約予定の作成のみに使用し、Google API Services User Data Policy（Limited Use を含む）に準拠します。

4. 第三者提供・委託

法令に基づく場合を除き、本人の同意なく第三者に提供しません。メール送信・決済・ホスティング等の外部サービスに必要な範囲で委託する場合があります。

5. 保管・セキュリティ

取得した情報は適切に管理し、アクセストークン等は暗号化して保管します。

6. 開示・訂正・削除

本人からの請求により、保有個人データの開示・訂正・削除に対応します。
