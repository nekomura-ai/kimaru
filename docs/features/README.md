# C. 実装優先順位（次にやる残作業）

実装済みを除いた残作業を優先順に並べる。上から着手する。

1. 🔴 **受付期間 無料2ヶ月の反映**（[05](./05-booking-range.md)）— 小さく着手しやすい。
2. 🔴 **事前アンケートのゲスト動的表示＋回答保存**（[10](./10-questionnaire.md)）— コア機能で半分実装済み。
3. 🔴 **プロフィールのサーバ保存**（[17](./17-profile.md)）— リマインダーの前提。
4. 🔴 **22分前リマインダー（プロフィール付き）**（[21](./21-reminder.md)）— 差別化の肝。
5. 🟡 **複数の予約ページ対応（無料2/有料・猫5）**（[24](./24-multiple-booking-pages.md)）。
6. 🟡 **予約完了メールの独自化**（[11](./11-notification-email.md)）。
7. 🟡 **占い的インサイトの高度化・生年月日「非公開」対応**（[16](./16-birthday.md)）。
8. 🟡 **運営コンソール拡充**（全ユーザー一覧・Cat Key 承認制）（[22](./22-admin-console.md)）。
9. ⚪ **会員同士の相互質問**（[20](./20-member-mutual-questions.md)）。
10. ⚪ **AIアシスト高度化（LLM 連携）**（[18](./18-ai-assist.md)）。
11. ⚪ **将来構想**: 無料お試し期間 / AI検索・顧客検索 / 議事録連携（マスタープラン、[23](./23-meeting-minutes.md)）/ 法人プラン / Zoom自動発行（[13](./13-plans.md) / [14](./14-customer-management.md) / [06](./06-location-type.md)）。
