# プライバシーポリシー

> 本文は事業者回答（2026-06-09）を反映済み。Google OAuth 審査では本ポリシーを**公開URL**で掲載する必要がある。

最終更新: 2026年6月9日

一般社団法人ぴんころ（以下「当社」）は、日程調整サービス「キマル」（以下「本サービス」）における利用者の個人情報を、以下の方針に基づき取り扱います。

## 1. 取得する情報

- **アカウント情報**: 氏名、メールアドレス、ログイン方式（Google アカウント／メール+パスワード）。
- **Google 連携情報**: Google ログイン時に取得するプロフィール（メール・氏名）、および連携を許可した場合の **Google カレンダーの空き時間・予定情報**、カレンダー予定の作成・Google Meet リンク発行に必要な情報。
- **予約情報**: 予約日時、開催方法、事前アンケートの回答、生年月日（任意・非公開選択可）。
- **利用記録**: 面談ログ、相手管理上のメモ・評価、アクセスログ、Cookie・セッション情報。
- **決済情報**: 有料プラン利用時の決済関連情報（決済代行事業者を通じて処理。当社はカード番号を保持しない）。

## 2. 利用目的

- 本サービスの提供（日程調整、カレンダー連携、予約管理、通知）。
- 事前アンケート・生年月日に基づく関係構築支援（インサイト表示等）。
- 不正利用の防止、招待コード（Cat Key）の適正運用。
- サービス改善・統計分析（個人を特定しない形での質問傾向の集計等）。

## 3. 第三者提供・外部サービス

本サービスは以下の外部サービスを利用し、目的の範囲で必要な情報を連携します。

- **Google（Google OAuth / Calendar / Meet）**: ログイン認証、カレンダー連携、予定・Meet 発行。
- **Supabase**: データベースおよびデータ保管。
- **Square**: 有料プランの決済処理（カード情報は Square 側で処理。当社はカード番号を保持しない）。

## 4. Google ユーザーデータの取り扱い（Limited Use）

本サービスによる Google API から取得した情報の利用・転送は、[Google API Services User Data Policy](https://developers.google.com/terms/api-services-user-data-policy)（Limited Use 要件を含む）に準拠します。カレンダー情報は本サービスの機能提供のためにのみ使用し、広告目的では利用しません。

## 5. 保管期間・削除

- 取得した情報はサービス提供に必要な期間保管します。
- 利用者はアカウント削除を申請でき、削除時に関連情報を適切に消去します（法令上保管が必要なものを除く）。
- Google 連携はいつでも解除できます。

## 6. 利用者の権利

開示・訂正・削除・利用停止の請求が可能です。下記連絡先までご連絡ください。

## 7. Cookie

ログイン状態の維持（セッション）に Cookie を使用します。
