## 有料版でできるようになること（差別化ポイント）

無料版が「予約を決める」までを担うのに対し、有料版は **予約の前後** を強化します。

1. **相手をより深く理解できる**
   - 事前アンケートを最大5問・自由編集。生年月日からの占い的インサイトで、相手に合わせたアプローチ方法が分かる。
2. **当日の質を上げる**
   - 22分前に相手のプロフィール付きリマインダー。会う前に相手を知り、距離を縮められる。
3. **関係を続けられる**
   - 顧客管理・予約履歴・面談メモ・印象スコアで、会った相手を資産として管理。
4. **自分を魅力的に見せられる**
   - 高度なプロフィールシート（色・太字・見出し。画像は今後）と公開プロフィールページ（`/u/<slug>`・誰でも閲覧可）で、押しやすく綺麗に共有。
5. **長く先まで予約を受けられる**
   - 公開範囲が6ヶ月先まで（無料は2ヶ月先まで）。
6. **AIで次の一手まで自動化（プレミアムプラン）**
   - 相手データから、AIが次回アポのアプローチ・刺さる話題・避けたい入り方・次の一手を提案（GPT-5.4 Mini）。¥2,200/月・無料お試しなし・フェーズ2で開放。
