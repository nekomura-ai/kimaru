日程調整ツール「キマル」の無料版と有料版の違いをまとめた説明資料です。

> 凡例: ✅ 利用可 / − 非対応 / 🔜 提供予定（開発中・未リリース）

| | 無料版 | 有料版（Pro） |
|---|---|---|
| こんな人に | まず日程調整を無料で使いたい | 相手との関係づくり・顧客管理まで踏み込みたい |

## 有料版でできるようになること（差別化ポイント）

無料版が「予約を決める」までを担うのに対し、有料版は **予約の前後** を強化します。

1. **相手をより深く理解できる**
   - 事前アンケートを最大5問・自由編集。生年月日からの占い的インサイトで、相手に合わせたアプローチ方法が分かる。
2. **当日の質を上げる**
   - 22分前に相手のプロフィール付きリマインダー。会う前に相手を知り、距離を縮められる。
3. **関係を続けられる**
   - 顧客管理・予約履歴・面談メモ・印象スコアで、会った相手を資産として管理。誕生日メールも自動送信。
4. **自分を魅力的に見せられる**
   - 高度なプロフィールシート（色・画像・見出し）と「プロフィールはこちら」リンクで、押しやすく綺麗に共有。
5. **長く先まで予約を受けられる**
   - 公開範囲が6ヶ月先まで（無料は2ヶ月先まで）。
